# ConstructAI ERP Prototype

This is a prototype Enterprise Resource Planning (ERP) system tailored for the construction industry.

## Quick Start
1. **Server**: `cd backend && npm run dev` (Runs on port 5000)
2. **Client**: `cd frontend && npm install && npm run dev` (Runs on port 3000)
3. **Login**: `alice@construct.ai` / `password123`

## Features
- **Core ERP**: User Management, RBAC, Audit Logs.
- **Finance**: GL, Invoices (Multi-currency), Payments, Reports.
- **AI Insights**: Project Risk Analysis (Budget vs Progress Logic), Cash Flow Forecasting.
